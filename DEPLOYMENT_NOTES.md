# ملاحظات النشر - Nurzao Platform

## المشكلة الأساسية
زر إرسال الطلبات في نموذج الاتصال لم يكن يعمل بشكل صحيح، والبيانات لم تصل إلى البريد الإلكتروني `Nurzao.Ops@gmail.com`.

## الحل المطبق

### 1. تحديث Backend (app.py)
- تم إضافة نظام تسجيل شامل (Logging) لتتبع جميع العمليات
- تحسين معالجة الأخطاء في دالة `send_email()`
- التأكد من إرسال البريد الإلكتروني للعميل وفريق العمليات
- إضافة رسالة تأكيد تلقائية للعميل بعد إرسال الطلب

### 2. تحديث Frontend (index.html)
- تحسين دالة `handleContactSubmit()` مع:
  - التحقق من صحة البيانات المدخلة
  - بناء URL الـ API ديناميكياً بناءً على النطاق الحالي
  - معالجة أفضل للأخطاء مع عرض رسائل خطأ واضحة
  - تعطيل زر الإرسال أثناء معالجة الطلب لمنع الإرسال المتعدد
  - إعادة تفعيل الزر بعد انتهاء العملية

## خطوات النشر

### على PythonAnywhere أو أي منصة استضافة:

1. **سحب التحديثات من GitHub:**
   ```bash
   cd /path/to/nurzao-platform
   git pull origin main
   ```

2. **تثبيت التبعيات:**
   ```bash
   pip install -r requirements.txt
   ```

3. **تعيين متغيرات البيئة:**
   - تأكد من تعيين `SENDER_EMAIL` و `SENDER_PASSWORD` في إعدادات التطبيق

4. **إعادة تحميل التطبيق:**
   - في PythonAnywhere: اضغط على زر "Reload" في صفحة Web
   - في Heroku: استخدم `git push heroku main`

## اختبار التطبيق

### اختبار محلي:
```bash
python3 app.py
```

ثم افتح `http://localhost:8080` وحاول إرسال طلب.

### اختبار في الإنتاج:
1. افتح الموقع على `https://nurzao.com`
2. ملء نموذج الاتصال بالبيانات الكاملة
3. اضغط على زر "إرسال الطلب"
4. تحقق من:
   - ظهور رسالة نجاح
   - وصول البريد الإلكتروني إلى `Nurzao.Ops@gmail.com`
   - وصول رسالة تأكيد للعميل

## استكشاف الأخطاء

### إذا لم يعمل الزر:

1. **افتح أدوات المطور (F12) وتحقق من:**
   - Console للبحث عن رسائل الخطأ
   - Network لرؤية طلب الـ POST والاستجابة

2. **تحقق من سجلات الخادم:**
   - في PythonAnywhere: اذهب إلى "Web" > "Log files"
   - ابحث عن رسائل الخطأ المتعلقة بـ SMTP

3. **المشاكل الشائعة:**
   - **خطأ SMTP**: تأكد من أن كلمة المرور صحيحة وأن Gmail يسمح بالوصول
   - **CORS Error**: تأكد من أن الـ API يعمل على نفس النطاق
   - **Timeout**: قد يكون الخادم بطيئاً، حاول مرة أخرى
